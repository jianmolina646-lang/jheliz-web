help ¿Cómo funciona el enforcement?

  1. Activá tu dispositivo TOTP siguiendo los pasos de arriba.
  2. Verificá que tu app autenticadora muestra códigos de 6 dígitos.
  3. Guardá los códigos de respaldo en un lugar seguro.
  4. Cuando estés seguro de que funciona, en producción agregá a tu archivo .env: 
    ADMIN_2FA_ENFORCED=True
  5. Reiniciá Gunicorn para que tome el cambio. A partir de ahí, el login del admin pedirá tu código TOTP.

⚠ Importante: Si activás ADMIN_2FA_ENFORCED=True sin tener un dispositivo confirmado, te bloqueás fuera del admin. Por eso esta página se asegura de que primero el dispositivo esté confirmed antes de mostrarte las instrucciones.

Rescate por SSH si te bloqueas: python manage.py addstatictoken <tu_usuario> te genera un token de uso único para entrar.
